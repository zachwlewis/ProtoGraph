# ngsketch

Offline-first web app for quickly mocking up node graph screenshots.

## PR1 status
- Infinite canvas with pan/zoom and grid
- Add node from toolbar or double-click
- Drag/select nodes
- Delete selected node (`Delete` / `Backspace`)
- Duplicate selected node (`Ctrl/Cmd + D`)
- Unit tests for graph node mutations

## Run
```bash
npm install
npm run dev
```

## Test
```bash
npm test
```
